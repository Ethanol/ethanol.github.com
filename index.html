    邻域2022

    2022-12-28
    { 路 }

    男友问我，想不想给这三年的生活写个总结？我说，你怎么知道这一切就结束了？

    2022年对绝大多数人来说都是不太好过的一年。年初的盛世幻境尚未结束，就开始了漫长的似乎永无止境的封城、核酸和离大谱；到了年底，调转方向猛踩油门，又是另一种荒谬。康复后，今天久违地出门吃了顿饭，就像一场虚空大梦，不知今夕何夕。

    可能也是这一年里，再对政治冷感的人也感受到了，你不关心政治，政治天天关心你。终于有了勇敢的人群，终于有了愿意为了自己、周遭或远方呐喊的声音。有了这些激励，即便前路依然晦暗，但不再怀疑。

    今年生活没太多变化。三十岁了，但和之前好像也没什么区别。拿到了人生第一份正式工作，工作上忙了些、累了些，有些是自己想做的，也有些是自己不想做的。不太想被科研这套游戏规则规训，趁年轻做点有意思的事儿，说不定过几年也就不做了。当然，虽然是工作，但今年大概四分之一的时间都被困在家中，和男友每日大眼瞪小眼，竟还没有互相厌弃，堪称爱情神话。

    若说今年有什么感悟，最大的便是更加关心「邻域」。隔离日子太久，所以更加期待和三五好友的每一次真实的相聚；时势波动剧烈，所以更要打理好接下来三个月的三餐药品和每日心情。今年难得地回了两次家乡，参与了几件自己觉得麻烦但好玩的事儿，算是一点小的努力。宏大叙事在这种不确定下越来越成为了一个笑话，不再愿意相信那些遥远的美好的虚幻的未来。但这也不是「无力感」：多说点话，多参与点事情，保持敏感，在「邻域」之中互相影响，然后往外拓展一点点，让身边变好一点点，依然是可以做、值得做的。

    后天要出门做三年来的第一次旅行了，久违地对生活有了一些期待。未来一年，希望我的心和步伐能够把「邻域」一点点展开。

    去爱去哭去疑惑

    2022-10-20
    { 音乐 豆瓣 }

    豆瓣是我的文艺世界启蒙。上大学后不久，我就在上面听音乐、看乐评，标记了许多条目，也认识了一些人。在豆瓣上，有一位小有名气的用户叫「汤川」，我应该很久之前就关注了他。我记得他喜欢陈珊妮，我从他那里得到过很多关于陈珊妮的新鲜有趣的观点。今年一月份的时候，汤川还关注了我们的微信公众号，我高兴了好一阵——尽管按照微信公众号现在的生态和我们的点阅率，他很有可能就是随手关注了一下就再也不会点开，但是我还是觉得，说不定他哪天还是会看到我写的东西。我一度很爱写东西、却总是觉得自己水平不够，有这么一位景仰的同辈能关注我，已经足够开心了。

    前两天我偶尔在社交网络上看到了「汤川」自杀的消息。我在想，这位「汤川」是不是我知道的汤川。打开豆瓣却找不到汤川的账号，后来才得知，他的号被封了。这两天陆续看到了两篇纪念他的文字，才知道，他确实离开了。在这两篇文字里，提到了汤川曾建过音乐群，也是在那里逐渐与他熟悉；我依稀记得我也在他的豆瓣上看到过群的二维码，但怯于社交的我，并没有敢加进去。

    然后我很难过。我并不算认识汤川，但是我还是很难过。互联网让我们擦肩而过，但等我回头之时，那些为数不多的痕迹早就被暴力地抹去了——更可惜的是，那些闪耀着人性和智慧的东西，也通通被抹去了。随着年纪一点一点长大，对生活的热爱一寸一寸退潮，还留恋的东西很少，能记得的东西也不多。但我一直欣赏那种直觉般的敏感和对事物毫不迟疑的热爱，当遇到这样的人时，我会不自觉把自己投射上去，虽然我自己身上这两样东西已所剩无几。这些特质就像无瑕的水晶花瓶，要么被收纳在角落里布满尘土，要么晶光灿烂地走向破碎。看到它破碎了，我很难过。

    今天午休，听了小老虎、也是福和周士爵的新专辑《去爱去哭去疑惑》。这张专辑温暖、熨帖又伤感，让我慢慢松弛下来。半梦半醒之间，听到专辑最后《一笑置之》里童漠男的脱口秀：

    我跟我爸去吃肯德基
    我爸能觉得劲脆鸡腿堡很伤感
    他说漠男你看，都是鸡腿堡
    无非它是比香辣的那个少了点儿辣味儿
    它就没有香辣鸡腿堡受人欢迎
    这是不是非常的伤感？

    然后趴在桌子上开始哭。有一阵子没哭过了，但这一阵子，我非常的伤感。
